# Buyer FAQ

Can the system run on private infrastructure?

Yes. The project supports full self-hosting.

Does the system respect robots.txt?

Yes. robots.txt is validated before each request.

Can ingestion be restricted by domain?

Yes. Domain rules are defined through YAML policy files.

Is sensitive data stored?

No. PII scrubbing removes sensitive patterns before storage.

Can audit logs be exported?

Yes. Audit logs can be exported from the dashboard.

Is the system designed for AI data ingestion?

Yes. The architecture supports structured ingestion pipelines suitable for AI and analytics workloads.
