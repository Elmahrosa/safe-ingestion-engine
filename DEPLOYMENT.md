# Contributing

Thank you for your interest in Safe Ingestion Engine.

## Development Setup

```

make dev

```

Run tests

```

pytest

```

---

## Pull Requests

All pull requests must:

• pass CI tests  
• include tests for new functionality  
• follow existing code style  

---

## Reporting Issues

Please include:

• reproduction steps  
• logs if available  
• expected vs actual behavior
