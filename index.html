Core Guarantees

✓PII auto-redacted — email, phone, SSN, IPv4, credit card via regex + Presidio NLP
✓SSRF impossible — async DNS + connect-time revalidation + redirect filter
✓robots.txt structural enforcement — PolicyEngine before every fetch, never advisory
✓YAML per-domain rules — allow/deny, path blocking, crawl budgets, delay controls
