// Why AI Ingestion Gateway
Built for compliance from day one.
🛡
SSRF Protection
Async DNS blocks private IPs, loopback, cloud metadata endpoints before any outbound fetch. DNS rebinding attacks blocked.
🧼
PII Auto-Scrubbing
Email, phone, SSN, IPv4, credit card — HMAC-hashed and redacted before data leaves the pipeline. Zero PII in storage.
📜
robots.txt Enforcement
Checked structurally before every fetch — not advisory. Blocked requests cost zero credits. Crawl budget enforced per domain.
⚙️
YAML Policy Rules
Per-domain allow/block rules, crawl budgets, path controls, concurrency limits — all declarative YAML config.
⚡
Async Job Queue
FastAPI + Celery + Redis. Submit a job, get a job ID in milliseconds. Poll for results. Auto-retry with backoff.
🤖
MCP + x402 Ready
Works as an MCP tool in Claude, Cursor, Windsurf. AI agents pay per-request via USDC on Base — zero signup needed.
Quick Start
# 1. Submit ingestion job
curl -X POST https://safe.teosegypt.com/v1/ingest_async \
  -H "X-API-Key: aig_YOURKEYHERE" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com"}'

# → {"job_id": "abc123", "status": "PENDING"}

# 2. Poll for result
curl https://safe.teosegypt.com/v1/jobs/abc123 \
  -H "X-API-Key: aig_YOURKEYHERE"

# → {"status":"COMPLETED","pii_found":2,"result_excerpt":"..."}
